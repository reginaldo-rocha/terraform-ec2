## 📌 Visão Geral

Este projeto demonstra a criação de uma instância EC2 na AWS utilizando Terraform, aplicando conceitos de Infraestrutura como Código (IaC).

O objetivo é mostrar, de forma prática, como automatizar a criação de recursos na nuvem.

---

## 🧱 Arquitetura

Terraform → AWS Provider → EC2 (Amazon Linux)

---

## ⚙️ Tecnologias Utilizadas

- Terraform
- AWS (EC2)
- AWS CLI
- VS Code

---

## 📁 Estrutura do Projeto
